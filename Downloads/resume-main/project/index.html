Project Name: Grade Management
Personal project

Purpose:

It is to help students easily manage the grade themselves. I always have some trouble finding the percentage of the assignments or exams and calculating grade every time to know where I am standing.

Skill:

Amazon Cognito(signup, login), React js (front-end), node express (back-end and api)

Content:

It allows students to create semester and courses for each semester and tasks for each course. When students create tasks(ex. programming as), students can write what percentage the task contribute to the course grade(40% among 100%) and how many sub-tasks(3 programming as) it has. When submitting each grade for subtasks, it calculates the grade for the tasks, course and semester. It assume non-filled parts (80% tasks filled but 20% tasks not filled => calculate as 100% for rest 20%).
